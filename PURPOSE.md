# Purpose

This directory appears to be a research workspace for studying learning problems connected to theorem proving and formal reasoning.

Based on the current contents, its purpose is to collect:

- topic areas such as `theorem_provers`, `formal_verification`, `phase_transitions_in_learning`, and `learning_and_submodularity`
- notes that connect learning dynamics to ideas like concavity, diminishing returns, and submodularity
- small executable experiments that test those ideas with toy RL simulations
- generated plots and run instructions that support those experiments

At the moment, the active work is concentrated in `alphaproof/`, which contains:

- conceptual notes on learning, compute, concavity, and submodularity
- Python scripts for toy reinforcement-learning experiments
- generated visualization outputs
- a run guide for reproducing those outputs

So the practical purpose of this repository, as it currently stands, is:

to explore and document how learning behavior scales with compute, especially in theorem-proving or AlphaProof-like settings, using a mix of research notes and small simulation artifacts.
